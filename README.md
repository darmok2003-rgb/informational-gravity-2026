## Abstract

We present a unified action functional that integrates the Einstein-Hilbert gravitational action, Stueckelberg-modified scalar electrodynamics (γ = 1), and quantum informational thermodynamics. By elevating the Von Neumann entropy \( S_{vN}(\rho) \) to a dynamical variable via a Lagrange multiplier λ (effective informational temperature), we derive an explicit informational stress-energy tensor \( T^{\text{info}}_{\mu\nu} = \lambda S_{vN}(\rho) g_{\mu\nu} \). In the macroscopic coherent limit, the framework reduces to an Entropic Gross-Pitaevskii equation with exact logarithmic stabilization term \( \lambda \ln |\psi|^2 \), supporting stable non-dispersive Gaussons as topological dark matter candidates.

The model naturally generates dynamic dark energy from quantum informational dissipation and suppresses singularities via information-density-dependent back-reaction (respecting the Bekenstein bound). Predictions are testable via cryogenic BEC interferometry and galactic rotation curve analysis.
